# Public Core Maintenance

This repository is the canonical public home for product-neutral LATTICE
models, operators, registry contracts, storage behavior, and graph boundaries.

## Promotion boundary

Related private incubation may explore customer deployments, funding,
organization-specific integrations, approval custody, and protected evidence.
That material is not part of the public core.

A change may move into this repository only when it:

1. is useful without private context;
2. contains no customer, employee, credential, approval, or protected source
   material;
3. uses product-neutral names and contracts;
4. includes public tests and documentation;
5. passes the repository's formatting, lint, and test gates.

Private history must never be merged, rebased, or force-pushed into this
repository. Promote a reviewed change as a new public commit with fresh public
authorship metadata. Public-core fixes may be ported back into private
incubation, but the public API remains the compatibility reference.
